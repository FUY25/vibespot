# VibeSpot Install

1. Build from source with `./scripts/dev-run.sh`, or download `VibeSpot.app` if you publish a release.
2. Move `VibeSpot.app` to `/Applications` when using a packaged build.
3. Launch once and allow any macOS trust prompts.
4. Complete onboarding.
5. Verify the first action works: run `new codex` from the search panel.
